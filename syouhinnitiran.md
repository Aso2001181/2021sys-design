# 商品一覧＆詳細

```uml
@startuml
ユーザー -> webサーバー:商品一覧
webサーバー -> DBサーバー:商品一覧
DBサーバー -> DBサーバー:一覧情報
DBサーバー -> webサーバー:一覧情報
webサーバー -> ユーザー:一覧情報
ユーザー -> webサーバー:商品詳細
webサーバー -> DBサーバー:商品詳細
DBサーバー -> DBサーバー:詳細情報
DBサーバー -> webサーバー:詳細情報
webサーバー -> ユーザー:詳細情報
alt カートに入れる
ユーザー -> webサーバー:商品をカートへ
webサーバー -> DBサーバー:商品をカートへ
DBサーバー -> DBサーバー:商品をカートに登録
DBさーばー -> webサーバー:カート内に追加
webサーバー -> ユーザー:カート内表示
else 前の画面へ戻る
ユーザー -> webサーバー:前の画面情報
webサーバー -> DBサーバー:前の画面情報
DBサーバー -> DBサーバー:前の画面の情報
DBサーバー -> webサーバー:前の画面へ
webサーバー -> ユーザー:画面表示
end
@enduml
```
